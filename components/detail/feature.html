Custom Tag implementation placed at app/liquid/tags/feature.rb

In the anadea.ifo project

Usage example

{% feature subtitle: '$', title: '331' %}
thousands projected e Learning market size by 2025
{% endfeature %}

Generated HTML:

<div class="feature ">
  <div class="feature__mark">
    <span class="feature__subtitle">$</span>
    <h3 class="feature__title">331</h3>
  </div>
  <div class="feature__description">
    thousands projected e Learning market size by 2025
  </div>
</div>

Accepts attributes separated by comma:

  • title: 'title value'

  • subtitle: 'subtitle value'

  • icon: '/path/to/icon_file'

Variable declaration order determines the order of the containers

If the subtitle: is first

{% feature subtitle: '$', title: '331' %}
thousands projected e Learning market size by 2025
{% endfeature %}

Generated HTML:

...
<div class="feature__mark">
  <span class="feature__subtitle">$</span>
  <h3 class="feature__title">331</h3>
</div>
...

If the title: is first

{% feature title: '331', subtitle: '$'  %}
thousands projected e Learning market size by 2025
{% endfeature %}

Generatet HTML:

...
<div class="feature__mark">
  <h3 class="feature__title">331</h3>
  <span class="feature__subtitle">$</span>
</div>
...

Using icon:

{% feature icon: '/path/to/icon_file'  %}
thousands projected e Learning market size by 2025
{% endfeature %}
